⏳ Life Clock — Time Since Birth

A live age calculator that shows exactly how long you've been alive — down to the second.

Enter your date of birth and watch a real-time counter track:
- Years, months, and days (calendar-accurate)
- Total hours, minutes, and seconds since you were born

✨ Features

- Real-time counter that updates every second
- Calendar-accurate breakdown (years / months / days)
- Cumulative totals (total hours / minutes / seconds elapsed)
- Loading states on page load and on calculation
- Clean, dark, distraction-free interface
- Fully responsive — works on mobile and desktop
- No backend, no dependencies — pure HTML, CSS, and JavaScript

🛠️ Built With

- HTML5 — structure
- CSS3 — styling, animations, and layout
- JavaScript (Vanilla) — date calculations and live updates
